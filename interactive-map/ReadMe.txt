
1. The interactive map styling code is written in "default.css" which is compiled from scss, which are placed in the scss folder

2. The Javascript code is written in "interactive-map.js" which bind all the events

3. The following function are involved in this

    i) "bind_maps" binds all the events

    ii) "drawPins" draw the pins over the image

    iii) "fetchPlaceData" call when user click on the marker and find it in the json then show details



If you make any change in JSON then you have to update it's properties names in

"drawPins" and "fetchPlaceData"

should be updated.
